API

Nos proponemos poner en abierto toda nuestra sección de la biblioteca y poner a tu disposición nuestra API con toda la documentación sobre plantas. Esta API, continúa en desarrollo y en breve proporcionaremos el servicio para obtener tu Api-Key. Continuamos con el trabajo para ponder habilitar esta función lo antes posible!

GET
/plants/:number/:page/:order/:sort Devuelve un número de elementos de una página especificada.
  • number: número de elementos por página.
  • page: página.
  • order: ordenar por nombre o familia de la planta (NAME o FAMILY).
  • sort: ordenar descendentemente, por defecto ascendente (DESC).

        "family": number,

        "photo": "string",
        "name": "string"
